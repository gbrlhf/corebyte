# Corebyte

Corebyte merupakan project landing page yang dibuat untuk memenuhi tugas mata kuliah Antarmuka Manusia dan Mesin. Website ini menampilkan tampilan modern dan responsif untuk produk gaming peripheral dan audio.

## Fitur
- Desain responsif untuk desktop dan mobile
- Tampilan modern dan minimalis
- Hero section dengan image slider
- Showcase produk
- Section testimonial
- Smooth scroll animation
- Form newsletter
- Tombol WhatsApp floating

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript

## Struktur Project
```bash
corebyte/
│
├── index.html
├── img/
│   ├── mouse.png
│   ├── keyboard.png
│   ├── iem.png
│   ├── headset.png
│   └── DAC.png
```

## Cara Menjalankan
1. Download atau clone repository ini
2. Buka folder project
3. Jalankan file `index.html` di browser

## Tujuan Project
Project ini dibuat sebagai bagian dari tugas mata kuliah Antarmuka Manusia dan Mesin untuk mempelajari desain antarmuka dan pengembangan website responsif.
