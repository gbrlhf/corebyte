# Corebyte

Landing page toko peripheral gaming dan audio, dibuat untuk tugas mata kuliah Antarmuka Manusia dan Mesin.

## Deskripsi

Corebyte menampilkan produk unggulan, testimoni pelanggan, dan informasi brand dalam satu halaman yang modern dan responsif.

## Teknologi

- HTML5, CSS3, JavaScript (Vanilla)
- CSS Grid & Flexbox untuk layout
- Google Fonts (Inter, DM Serif Display)

## Fitur

- Hero section dengan image slider otomatis
- Horizontal swipe produk di mobile
- Animasi scroll reveal
- Responsif (desktop, tablet, mobile)
- Form newsletter & tombol WhatsApp floating

## Struktur

```
corebyte/
├── index.html
└── img/
    ├── mouse.png
    ├── keyboard.png
    ├── iem.png
    ├── headset.png
    └── DAC.png
```

## Cara Menjalankan

Buka `index.html` langsung di browser — tidak perlu instalasi apapun.
